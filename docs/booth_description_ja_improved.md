おちびちゃんズ化ツール（Ochibi-chans Converter Tool）

Hierarchyで選択した改変済みアバターを、ワンクリックで「おちびちゃんズ」化できるUnityエディタ拡張です。
このツールは、改変済みアバターをおちびちゃんズ化したい人向けの商品です。

●「おちびちゃんズ」とは？
おちびちゃんズは、夕時茶屋様が制作・提供する商品で、
さまざまなアバターたちの表情や体格をおちびな姿に調整したプレハブ差分データです！

【このツールの強み】
・右クリックからすぐ実行：作業導線がシンプル
・コピーしてから変換：元アバターを残したまま試せる
・対応範囲が広い：人気アバター＆おちびちゃんズに対応

⚠️ 本ツールは非公式（ファンメイド）です。
夕時茶屋様が制作・提供する公式ツールではありません。

【インストール（VCC）】
1. 以下リンクを開き、VCCへリポジトリを追加
https://aramaa-vr.github.io/vpm-repos/redirect.html

2. VCCの「Settings → Packages → Installed Repositories」で「aramaa」が有効になっていることを確認

3. 対象プロジェクトの「Manage Project」で
「おちびちゃんズ化ツール（Ochibi-chans Converter Tool）」を追加

※ 「Modular Avatar」「lilToon」などの依存パッケージは自動で導入されます。

【使い方】
1. UnityのHierarchyで変換元アバターを右クリック
2. 「Aramaa → おちびちゃんズ化ツール（Ochibi-chans Converter Tool）」を選択
3. 変換先のおちびちゃんズをプルダウンで選択
4. 「実行（コピー→変換）」をクリック
