# おちびちゃんズ化ツール
（Ochibi-chans Converter Tool）

VRChatアバターを **「おちびちゃんズ」に変換**する Unity Editor 拡張です。  
**元アバターを壊さない（Ctrl+D相当で複製 → 変換）** 方式です。

<div class="hero-actions">
  <a class="btn btn-primary" href="https://aramaa-vr.github.io/vpm-repos/redirect.html" target="_blank" rel="noopener noreferrer">➕ Add to VCC</a>
  <a class="btn" href="{{ "/install/" | relative_url }}">📦 導入</a>
  <a class="btn" href="{{ "/usage/" | relative_url }}">🧸 使い方</a>
  <a class="btn" href="{{ "/support/" | relative_url }}">📋 対応アバター</a>
</div>

> **「おちびちゃんズ」** とは、[夕時茶屋](https://okiyudachi.booth.pm/)様が制作・提供するさまざまなアバターたちの表情や体格をおちびな姿に調整したプレハブ差分データです！  
> ⚠️ 本ツールは 非公式（ファンメイド）ツールです。夕時茶屋様が制作・提供する公式ツールではありません。  

</div>

<div class="notice-callout">
  <strong>できること</strong>
  Hierarchyのアバターを複製し、変換先おちびちゃんズの設定（Scale / FX / Expressions / ViewPosition 等）を上書きします。
</div>
